



<l>CSS (Cascading Style Sheets): The language used to style and layout HTML documents.</l>

<l>CSS Grid: A two-dimensional layout system in CSS that provides more control over how web content is arranged.</l>

<l>Cross-Browser Compatibility: Ensuring a website works correctly on different browsers and devices.</l>

<l>CSS Reset/Normalize: A set of rules that remove browser inconsistencies in default styling.</l>

<l>DNS (Domain Name System): Translates domain names into IP addresses.</l>

<l>DOM (Document Object Model): The programming interface for web documents, allowing dynamic changes to HTML and CSS.</l>

<l>Domain Name: The address used to access a website.</l>

<l>Element: The basic building blocks of HTML, made up of tags and content.</l>

<l>Em (Unit): A relative CSS unit used to define size, based on the font size of the element.</l>

<l>Event: An action that occurs in the browser, such as a click or key press.</l>

<l>Event Listener: A JavaScript method used to handle events like clicks and keypresses.</l>

<l>Flexbox: A layout model that allows elements to be aligned and distributed within a container.</l>

<l>FTP (File Transfer Protocol): A method of transferring files between computers over a network.</l>

<l>Full Stack Development: Involves both front-end (client-side) and back-end (server-side) development.</l>

<l>Function: A block of code designed to perform a task in programming.</l>

<l>Git: A version control system that helps developers track changes to code.</l>

<l>HTML (Hypertext Markup Language): The standard language used to create web pages.</l>

<l>HTTP/HTTPS: Hypertext Transfer Protocol, the foundation of any data exchange on the Web. HTTPS adds security via encryption.</l>

<l>ID: A unique identifier for an element in HTML and CSS.</l>

<l>Inline Element: An element that does not start on a new line and only takes up as much width as necessary.</l>

<l>IP Address: A unique numerical address identifying a device on a network.</l>

<l>JavaScript: A programming language used to create dynamic and interactive effects within web browsers.</l>

<l>Media Query: A CSS feature used to apply styles based on device characteristics, like screen width.</l>

<l>Metadata: Information about a webpage, such as the title and description, typically stored in the head section of an HTML document.</l>

<l>Mobile-First Design: Designing web content first for mobile devices, then enhancing for larger screens.</l>

<l>Padding: Space between the content and the border of an element in the CSS box model.</l>

<l>Property: Defines the aspect of an element to be styled in CSS.</l>

<l>Pseudo-Class: A keyword added to a selector in CSS that specifies a special state of the selected element.</l>

<l>Pseudo-Element: A keyword added to a selector in CSS that allows you to style parts of an element.</l>

<l>Responsive Design: A design approach where web content adjusts to various screen sizes.</l>

<l>Script: A block of code that can be used to add functionality to a webpage, often written in JavaScript.</l>

<l>Selector: A CSS statement used to target HTML elements for styling.</l>

<l>Semantic HTML: HTML that conveys meaning about its content beyond just its presentation.</l>

<l>Server-Side: Refers to actions or scripts that run on a web server.</l>

<l>SSL (Secure Sockets Layer): A protocol for encrypting information sent between a browser and a web server.</l>

<l>Tag: Defines the start and end of an HTML element.</l>

<l>URL (Uniform Resource Locator): The address used to access a resource on the internet.</l>

<l>User Agent: Information about the browser, operating system, and device, often sent to web servers.</l>

<l>Variable: A container for storing data values in programming.</l>

<l>Version Control: A system that records changes to a file or set of files over time.</l>

<l>Viewport: The visible area of a web page on a device.</l>

<l>Web Hosting: A service that allows organizations and individuals to post a website on the internet.</l>

<l>Web Server: A computer that hosts websites and serves them to users' browsers.</l>

<l>White-Space: In CSS, controls the display of spaces, line breaks, and tabs.</l>

<l>XML (eXtensible Markup Language): A language designed to store and transport data.</l>
